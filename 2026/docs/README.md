# 文档目录

> v5.20.0 重组：按受众/用途分子目录，便于快速定位。

| 子目录 | 面向 | 内容 |
|--------|------|------|
| [`user/`](user/) | 终端用户 | 引导模式操作手册、流程文件（含 mermaid 流程图） |
| [`notes/`](notes/) | 终端用户 | 使用说明书、打包说明 |
| [`dev/`](dev/) | 开发 / 接手 AI | AI 交接文档、AGENTS 模板、程序与脚本清单、GitHub 上传说明 |
| [`design/`](design/) | 产品 / 架构 | 配置参考、NotebookLM 预设、增量同步设计、产品需求、任务清单 |
| [`plans/`](plans/) | 规划 | 进行中/近期规划与决策记录 |
| [`test-reports/`](test-reports/) | QA | 测试报告总览与历史快照 |
| [`archive/`](archive/) | 归档 | 已落地或过期的规划文档（只读，历史参考） |
| [`cursor-rules-templates/`](cursor-rules-templates/) | IDE | Cursor/Codex 模板规则（可选复用到其他项目） |

## 快速入口

- **我是新用户** → [user/操作手册-引导模式.md](user/操作手册-引导模式.md) 或 [notes/使用说明书.md](notes/使用说明书.md)
- **我是接手的 AI / 开发** → [dev/AI_交接文档_下一阶段开发.md](dev/AI_交接文档_下一阶段开发.md)
- **我要查配置键或预设** → [design/CONFIG_REFERENCE.md](design/CONFIG_REFERENCE.md)
- **我要查回归测试状态** → [test-reports/TEST_REPORT_SUMMARY.md](test-reports/TEST_REPORT_SUMMARY.md)
