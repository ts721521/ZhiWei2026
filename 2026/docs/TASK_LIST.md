# 任务清单（Roadmap）

## Phase 1：高价值低风险（当前主线）
- [x] 运行模式与配置分组重构（GUI 分区、模式联动、按模式保存）
- [x] 转换运行页二次分层（核心转换 / 过滤策略 / AI导出 / 增量同步）
- [x] 配置中心分层补齐（新增 AI 默认配置 / 增量默认配置）
- [x] 配置中心二次拆分（AI: 文本清单/结构向量；增量: 扫描重命名/去重打包）
- [x] Shared Defaults 再拆分（路径 / 进程策略 / 日志输出）
- [x] Merge Defaults 再拆分（行为默认 / 产物默认）并补充“配置中心=持久化默认”提示
- [x] Rules & Keywords 再拆分（排除规则 / 关键词策略）
- [x] 配置中心分区级重置（每个配置页支持“重置本分区”，不自动保存）
- [x] 配置中心未保存状态提示（默认值变更后显示“尚未保存”）
- [x] 保存后未保存状态精确回算（`Save Current Mode` 按配置差异自动判断是否仍有未保存项）
- [x] 配置分区级未保存标记（有改动的配置 Tab 名称后显示 `*`）
- [x] 未保存分区列表提示（配置中心顶部显示具体未保存分区）
- [x] 快速跳转未保存分区（按钮一键定位到首个未保存配置 Tab）
- [x] 未保存分区可点击跳转（点击分区名直接切换到对应配置 Tab）
- [x] 仅保存未保存分区（按分区差异写回 config，减少全量保存干扰）
- [x] 未保存分区保存结果反馈（显示本次已保存分区列表/无改动提示）
- [x] 分区级单独保存（每个配置分区提供“保存本分区”，减少跨分区误覆盖）
- [x] 恢复未保存分区（按分区回滚到 config 基线，支持已恢复分区反馈）
- [x] 未保存分区操作按钮计数（保存/恢复按钮动态显示待处理分区数量）
- [x] 恢复未保存分区二次确认（执行前确认回滚范围）+ 回滚前基线刷新
- [x] 回滚确认可配置（UI 默认项支持关闭“恢复未保存分区”二次确认）
- [x] 合并索引增强（Excel 可点击定位、MD5/页码信息）
- [x] 生成 `corpus.json`（统一语料清单）
- [x] 在 GUI 显示“本次产物摘要”（可选）

## Phase 3：增量同步与去重（运维 P0，分步上线）
- [x] **状态账本**: 实现 `FileRegistry` 类 (含 Path Normalize)。

- [x] `Global MD5`: 维护运行时 MD5 集合，先做同类型输出去重。
- [x] **变更检测**: 实现增量扫描 Added/Modified/Renamed/Unchanged/Deleted 统计，并落盘到账本。
- [x] **增量包**: 实现 `Update_Package` 生成（包含增量 PDF + 增量索引 + 增量 manifest）。
- [x] **GUI 适配**: 增量模式与去重开关（增量扫描/哈希校验/同名优先/MD5去重）。
